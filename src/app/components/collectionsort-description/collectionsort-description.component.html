Welcome to Collectionsort! This SPA is a data collection/management solution I created when I found I could not find a simple sortable table plugin, while learning Angular. Now I have used this solution in several enterprise applications and other projects. I first began with just the standard table, but then I needed to add sorting to the mix. This proved difficult and became even more challenging because I needed to add filtering as well. Collectionsort uses custom pipes including a pipe to render text in TitleCase, which capitalises the first letter of the string. Over time this solution has collected many small additions and custom edits to become capable of many more features. Written with custom pipes, services and fitlers, it is a very detailed and thorough Angular solution.

Collectionsort currently has two major divisions; Tablesort and Listsort. Tablesort is a table that is capable of the features listed below. Using JSON data it searches, sorts and filters based on the attributes displayed in each column of the table. Listsort is a much more condensed version of tablesort. It is simply a list that can be searched through and sorted through. This is good for listing items but only wanting to show one major attribute or attribute set. Please explore to learn more.
